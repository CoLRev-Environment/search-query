========================

**Error Code**: F1001

**Message**: ``Wildcard characters (*, ?, $) not supported in year search.``

**Scope**: PLATFORM.WOS

**Typical fix**: Replace with year range.

**Problematic query**:

.. code-block:: python

    A AND year=201*

**Correct query**:

.. code-block:: python

    A AND (year >= 2010 AND year < 2020)
