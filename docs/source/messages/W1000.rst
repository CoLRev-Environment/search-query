===========================

**Error Code**: W1000

**Message**: ``The value of NEAR operator is implicit``

**Scope**: PLATFORM.WOS

**Typical fix**: The parser automatically sets implicit NEAR values to the default of 15.

**Problematic query**:

.. code-block:: python

    A NEAR B

**Correct query**:

.. code-block:: python

    A NEAR/15 B
