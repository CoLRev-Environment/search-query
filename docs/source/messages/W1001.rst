===========================

**Error Code**: W1001

**Message**: ``Year span must be five or less.``

**Scope**: PLATFORM.WOS

**Typical fix**: The parser automatically sets the year span to 5.

**Problematic query**:

.. code-block:: python

    A AND PY=2000-2020

**Correct query**:

.. code-block:: python

    A AND PY=2015-2020
