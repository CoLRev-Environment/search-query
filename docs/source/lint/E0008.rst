=======================

**Error Code**: E0008

**Message**: ``The whole Search string is in quotes.``

**Scope**: all

**Problematic query**:

.. code-block:: text

    # PLATFORM.WOS
    "eHealth[ti] AND digital health[ti]"

**Recommended query**:

.. code-block:: text

    # PLATFORM.WOS
    eHealth[ti] AND digital health[ti]
