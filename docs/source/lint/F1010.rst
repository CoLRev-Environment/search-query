======================

**Error Code**: F1010

**Message**: ``Query contains invalid syntax``

**Scope**: all

**Problematic query**:

.. code-block:: text

    # PLATFORM.WOS
    eHealth[ti]

    # PLATFORM.PUBMED
    TI=eHealth

**Recommended query**:

.. code-block:: text

    # PLATFORM.WOS
    TI=eHealth

    # PLATFORM.PUBMED
    eHealth[ti]
