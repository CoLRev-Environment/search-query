**Error Code**: F2009

**Message**: ``Invalid DOI format.``

**Scope**: PLATFORM.WOS

**Problematic query**:

.. code-block:: text

    DO=12.1000/xyz

**Recommended query**:

.. code-block:: text

    DO=10.1000/xyz

**Typical fix**: Use a valid DOI format (e.g., starts with 10. followed by a numeric string and suffix).
